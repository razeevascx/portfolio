## Overview

This is the professional portfolio of **Rajeev Puri**, a Software Engineer specializing in modern web technologies. The site is designed as a sophisticated digital workspace, rooted in a **Cyber-Minimalist** aesthetic that emphasizes focus through high-contrast design, tactile textures, and atmospheric motion.

Built with **Next.js 16** and **React 19**, it serves as both a showcase of technical expertise and a functional platform for blogging, project management, and professional networking.

## Features

- **Cyber-Minimalist Design** — A dark-first aesthetic featuring dot-matrix patterns, topographic maps, and aurora light effects.
- **Dynamic Content Pipeline** — Seamless integration with **Notion** for blog posts and project data, combined with **MDX** for high-fidelity content.
- **Tactile Motion** — Atmospheric transitions and interactive UI elements powered by **Framer Motion**.
- **Live Integrations** — Real-time GitHub repository stats, verified **Credly** certifications, and **Cal.com** booking.
- **Modern Tech Stack** — Leveraging Next.js Turbopack, Tailwind CSS 4, and React Server Components for peak performance.

## Tech Stack

- **Frontend:** [Next.js 16](https://nextjs.org/) (App Router), [React 19](https://react.dev/), [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **CMS & Data:** [Notion API](https://developers.notion.com/), [MDX](https://mdxjs.com/), [GitHub API](https://docs.github.com/en/rest)
- **UI Components:** [Shadcn UI](https://ui.shadcn.com/), [Lucide Icons](https://lucide.dev/)
- **Integrations:** [Cal.com](https://cal.com/), [Credly](https://www.credly.com/)
- **Analytics:** [Vercel Analytics & Speed Insights](https://vercel.com/analytics)

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v20+)
- [Bun](https://bun.sh/) or [pnpm](https://pnpm.io/)

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/razeevascx/portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio
   ```

3. **Install dependencies**
   ```bash
   bun install

   ```

4. **Configure environment variables**
   Create a `.env` file in the root directory and add your credentials:
   ```env
   NOTION_TOKEN=your_token
   NOTION_DATABASE_ID=your_db_id
   GITHUB_TOKEN=your_token
   ```

5. **Start the development server**
   ```bash
   bun dev
   ```

6. **Verify installation**
   Open [http://localhost:3000](http://localhost:3000) in your browser.
